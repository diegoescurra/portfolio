# Proyectos

## Altomonte Bienes Raíces

Diego desarrolló una plataforma inmobiliaria para Altomonte Bienes Raíces.

El proyecto estuvo orientado a optimizar la gestión de propiedades y mejorar su visibilidad online.

Actualmente este proyecto no está en linea por petición de la agencia, ya que están cambiando su nombre.

## Ranut

Diego implementó una landing page para Ranut, una empresa de servicios y soporte para máquinas de café.

El objetivo del proyecto fue mejorar la comunicación de servicios y facilitar el contacto con clientes potenciales.

## Portteck

Diego desarrolló una aplicación móvil para Portteck, orientada a instaladores en terreno.

La aplicación centraliza información técnica de motores y sistemas de control de acceso para consulta operativa en tiempo real.
